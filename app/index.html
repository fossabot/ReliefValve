On the Main tab press the Choose button and pick a directory named "Steam" with the structure below to load the data about currently installed apps/games.

Under the heading "Steam App Settings" is a combo-box with the same options as the Steam client for auto-update behavior. The checkbox below it will apply the "offline fix" described [badly] here. Press Apply to save your chosen settings.

The blacklist is used to keep app/game data from being loaded.
